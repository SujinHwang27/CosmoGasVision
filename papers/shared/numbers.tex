% ----------------------------------------------------------------------
% SCHEMA (naming convention)
% ----------------------------------------------------------------------
% All macros use a hierarchical CamelCase scheme:
%
%   \<Statistic><Aggregator><Cell>
%
%   <Statistic>  = PF | KS | meanF | tauamp | Rsat | rrhoLog | meanFspread | ...
%   <Aggregator> = Bootstrap{Mean,Std,QLo,QHi} | SeedFortyTwo | (omit if single-value)
%   <Cell>       = Pone | Ptwo | Pthree | Pfour  (per-physics)
%                | (omit for cross-physics or single-cell quantities)
%                | Range / Lo / Hi for inclusive ranges
%
% Examples:
%   \KSBootstrapMeanPone        = 0.0357   (P1 KS bootstrap mean, [D-44])
%   \KSBootstrapStdPone         = 0.0024   (P1 KS bootstrap 1-sigma)
%   \KSBootstrapQHiPtwo         = 0.0758   (P2 KS q84 upper)
%   \PFresidPone                = 0.4155   (P1 P_F residual, seed=42 anchor)
%   \PFresidLo / \PFresidHi     = 0.3591 / 0.4155 (cross-physics range)
%   \meanFseedFortyTwoPone      = 0.97895  (P1 mean_F at seed=42 single-seed)
%   \RsatPone                   = 18.87    (P1 saturation-band R)
%
% Where a number has both seed=42 single-seed and 5-seed bootstrap forms
% the macro carries the aggregator explicitly. Seed=42 anchor values are
% load-bearing for tab:pub-t1-gates (the seed=42 anchor table). [D-44]
% bootstrap values are load-bearing for tab:headline-3gate.
%
% Constants (gate thresholds, observational anchors) are prefixed Gate or
% Anchor for legibility, e.g. \GateKS, \GatePF, \AnchorMeanF.
%
% Derived quantities (computed from primary values, e.g. cross-physics
% spread = 0.0064/0.9282 = 0.69%) are stored as their final rendered
% value to avoid \fpeval fragility; comments cite the derivation.
%
% Numbers in tab:pub-t1-gates are pinned to the seed=42 anchor by design
% ([D-44] explicit scope: P_F single-seed, KS/meanF reported bootstrap).
% Macro names carry the SeedFortyTwo qualifier where this matters.
% ----------------------------------------------------------------------
